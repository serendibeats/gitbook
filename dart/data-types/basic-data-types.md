# Basic data types

```dart
void main() {
  String name = 'Yongsun';

  int age = 12;
  double money = 69.99;
```

* Dart는 객체지향 언어이다.
* num은 int와 double의 부모 class이다.
* num type은 int 혹은 double 일 수 있다.
